# WP DB Surgeon

Lightweight WordPress database cleaner. 
No bloat, no paywalls, no direct SQL queries.

## Why another DB cleaner?
- WP-Sweep hasn't been updated since 2022
- WP-Optimize is bloated with cache/images features
- Advanced DB Cleaner locks scheduling behind $39/mo paywall

## Features
- Preview before delete — see exactly what will be removed
- Scheduled cleanup via WP-Cron (free)
- Uses native WP functions, not raw SQL
- Clean UI in WP 6.x style
